# Vaportext

For making **Ａｅｓｔｈｅｔｉｃ** text

## Usage

`go get github.com/JoelPagliuca/vaportext`

```
Usage of vaportext:
  -c    Copy output to clipboard
  -z    Run some Zalgo weirdness over the text
```
